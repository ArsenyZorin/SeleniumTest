При запуске проекта, не обязательно, но желательно, в обозревателе решений двойным щелчком левой кнопки мыши по AppulatebetaSelenium.cs открыть файл. И собрать проект.
Для этого, на вкладке "Сборка", выбрать "Собрать решение". Или воспользоваться комбинацией "горячих"клавиш: Ctrl+Shift+B.
Сборка -> Собрать решение.

Чтобы запустить тест, нужно зайти в обозреватель тестов. Для этого, в Visual Studio, на вкладке Тест, выбираем Окна и в выпадающем списке - Обозреватель тестов.
Тест -> Окна -> Обозреватель тестов.

В обозревателе тестов должен появиться элемент с названием: AppulatebetaSeleniumTest. Для его запуска, нужно нажать правой кнопкой мыши и выбрать: "Выполнить выбранные тесты".


Если тест не появился в "обозревателе тестов", то следует перестроить проект. Для этого: на вкладке Сборка, выбрать "Перестроить MuranoTestProject".
Сборка -> Перестроить MuranoTestProject

После этого, в "обозревателе тестов" должен появиться AppulatebetaSeleniumTest. 

В тесте проверяется обработка ввода корректных и некорректных данных. 
На первом этапе регистрации, для взаимодействия с пользователем, были добавлены несколько всплывающих окон, уведомляющих об ошибке.
На втором этапе регистрации, было использовано стандартное окно, которое появляется при попытке перехода не следующий этап, если не все данные были корректно введены

После выполнения теста, браузер автоматически закроется.
